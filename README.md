# Calculator App

Android Calculator Application

## Features
- Basic calculator (+, -, ×, ÷, %)
- Scientific calculator (sin, cos, tan, ln, log, √, ^, !)
- History records
- 17-digit input limit
- Thousand separators

## Tech Stack
- Java
- Android SDK 35
- AGP 8.3.0
- Gradle 9.2.1
- exp4j (math expression library)

## Build
```bash
./gradlew assembleDebug
```

APK output: `app/build/outputs/apk/debug/app-debug.apk`
